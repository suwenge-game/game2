# 新词挖掘系统使用指南

## 🚀 快速开始

### 1. 运行新词挖掘脚本
```bash
cd new_word_mining
python3 new_word_mining_script.py
```

### 2. 查看挖掘结果
- **JSON数据**: `new_words_*.json` - 包含所有关键词的详细数据
- **挖掘报告**: `mining_report_*.md` - 高潜力关键词摘要

### 3. 应用高潜力关键词
根据评分选择关键词：
- **评分7-10分**: 立即应用，创建相关内容
- **评分5-6分**: 观察测试，适度应用
- **评分<5分**: 暂时忽略，定期重新评估

## 🔧 自定义配置

### 修改数据源
编辑 `新词挖掘配置.json`:
```json
{
  "data_sources": {
    "google_trends": {
      "enabled": true,
      "keywords": ["your", "custom", "keywords"]
    }
  }
}
```

### 调整评估标准
修改评分权重:
```json
{
  "keyword_evaluation": {
    "search_volume_weights": {
      "high": 3,
      "medium": 2,
      "low": 1
    }
  }
}
```

### 添加新关键词
在脚本中添加基础关键词:
```python
self.base_keywords = [
    "your", "new", "keywords", "here"
]
```
